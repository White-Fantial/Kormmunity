# Project Brief — NZ Korean Community Market

## One-line Summary
뉴질랜드 한인 커뮤니티를 위한 카카오톡 친화형 지역 거래/질문/도움 게시판 웹사이트를 만든다.

## Background
현재 뉴질랜드 한인 커뮤니티의 가장 활발한 거래/정보 교류 채널은 카카오톡 오픈채팅방이다. 글을 올리고 대화하기는 편하지만, 오래된 글 검색, 카테고리별 탐색, 지역별 필터링, 판매 완료 관리, 신고/검토/제재 같은 구조화된 관리가 어렵다.

이 프로젝트는 카카오톡처럼 쉽게 글을 올리되, 웹사이트의 검색성과 분류성을 결합한 커뮤니티 마켓 플랫폼을 목표로 한다.

## Target Users
- 뉴질랜드 거주 한인
- 중고 물품을 사고파는 사람
- 지역 생활 질문을 올리는 사람
- 도움을 요청하거나 무료나눔을 하는 사람
- 커뮤니티 운영자, 코디네이터, 관리자

## Core Product Principles
1. 글 작성은 카카오톡처럼 가볍고 빠르게 한다.
2. 제목은 선택 사항이다. 제목 없이도 글을 작성할 수 있어야 한다.
3. 카테고리와 지역은 필수로 선택한다.
4. 목록에서는 지역과 카테고리 기준으로 쉽게 필터링한다.
5. 판매글은 가격과 판매 완료 상태를 지원한다.
6. 카카오 로그인 기반으로 가입/로그인을 간단히 한다.
7. 작성자에게 카카오톡으로 연락할 수 있는 흐름을 제공한다.
8. 운영진은 부적절한 글과 사용자를 단계적으로 관리할 수 있어야 한다.
